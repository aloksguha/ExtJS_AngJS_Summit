HTML Tags Basics

At the most basic level understand following are most popular HTML tags
  • Div – <div>
  • Span – <span>
  • Text Format – <b>, <>, <strong>
  • Paragraph – <p>
  • Headers – <h1> to <h6>
  • Table,Row, Column – <table><tr><td>col1</td><td>col 2</td></tr></table>
  • Anchor Tag – <a href=”….”></a>
  • Image Tag – <img src=”….”></img>
  • Line Break – <br>
  • Bullets – <ul>, <ol> and <li>
  • Forms – <form>
  • Form elements – <input type=”…”></input>, <textarea>, <select> etc
These elements can be distinguished between two type
  • Block Elements – Which take up a block of area on HTML, forcing next element to come on next line
  • Inline Elements- Which all elements to exist on the same line

CSS Primer

CSS properties can be declared in <style> tag with in the same HTML Page or in separate css files which are included as <link href=”special.css” rel=”stylesheet” type=”text/css”> CSS style can be applied in following manner

  • CSS style can be applied globally for a particular element 

    div { background:yellow}
    - This makes all div’s background yellow

  • CSS style can be applied to a CSS Class 

    .myclass {border:1px solid blue; }
    – This makes any element with class=”myclass” border as solid blue

  • CSS style can be applied to a Element Id 

    #myid { color: white ; }
    – This makes any element with id = “myid” text color to white

Topic 5 - Inheritance

Until you get out of your Java or C# Class paradigm, it is best to delay this topic. First of all JavaScript doesn’t have classes, it only have objects and a Function is a first class Object. The whole confusion is caused by the use of new keyword.

In JavaScript, Inheritance can be done is two ways

  • Classical - Mimicking a concept of Class in JavaScript
  • Modern/Prototypal – Leaving Classes out and inheriting from other Objects
