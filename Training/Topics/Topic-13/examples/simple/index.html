Simple Example

Code location: Training/Topics/Topic-13/examples/simple 

// Install required plugins. Plugins should be specified in package.json file.
$> npm install
...

// This will list Available tasks
$> grunt --help 
....

// Let's run concat task
$> grunt concat
Running "concat:dist" (concat) task
File "dist/simple.js" created.

Done, without errors.

Let's add new task

// Add grunt-contrib-jshint task
$> npm install grunt-contrib-jshint --save-dev

// add entry in Gruntfile to load the task "grunt.loadNpmTasks('grunt-contrib-jshint');"

// Lets run jshint
$> grunt jshint
