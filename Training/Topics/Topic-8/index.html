Understanding Media Queries

Media Query are CSS block which apply to the page based on a condition. Often these conditions are based on the width. In the following example, you will see if we resize the page, the email address on the left panel changes , it is more elaborate when the width is more and shrinks using email icons or nothing as width decreases. This is an example of Responsive Page. Where the Same Page looks proper be it a Desktop Browser, a Tablet Browser or a Smart Phone Browser. See Live Demo

Live Demo


HTML Code Snippet 

<div id="sidebar">
      <h3>Super team:</h3>

      <ul id="nav">
         <li><a data-email="chriscoyier@gmail.com" href="mailto:chriscoyier@gmail.com">Chris Coyier</a></li>
         <li><a data-email="isuredo@likeher.com" href="mailto:isuredo@likeher.com">Elisabeth Moss</a></li>
         <li><a data-email="marry@me.com" href="mailto:marry@me.com">Amanda Righetti</a></li>
      </ul>
</div>
CSS Code Snippet 
//Default CSS for the Email Link
#sidebar ul li a {
  color: #900;
  text-decoration: none;
  padding: 3px 0;
  display: block;
}
//CSS when the Width beyond 1001px
@media all and (min-width: 1001px) {
  #sidebar ul li a:after {
    content: " (" attr(data-email) ")";
    font-size: 11px;
    font-style: italic;
    color: #666;
  }

}
//CSS when the Width between 700px and 1000px
@media all and (max-width: 1000px) and (min-width: 700px) {
  #sidebar ul li a:before {
    content: "Email: ";
    font-style: italic;
    color: #666;
  }
}

//CSS when the Width between 520px and 699px or Beyond 1151px
@media all and (max-width: 699px) and (min-width: 520px), (min-width: 1151px) {
  #sidebar ul li a {
    padding-left: 21px;
    background: url(../images/email.png) left center no-repeat;
  }

}
The above example is taken for this Reference - http://css-tricks.com/css-media-queries/

Twitter BootStrap Purpose

Twitter BootStrap is best defined as a combination of

  • Boiler Plate/Scaffolding Layouts
  • Finished and Responsive CSS
  • Common and Good looking Components
  • Few Useful JavaScript Plugins
In short Twitter BootStrap will help you make Finished (New Age Cool Looking finished) Web Apps with Responsive UI.

Usage of Twitter BootStrap

To use Twitter bootstrap you will need to download it from http://twitter.github.com/bootstrap/
Create the following Project Structure 

  App/
  ├── index.html
  ├── css/
  │   ├── bootstrap.css
  │   ├── bootstrap.min.css
  ├── js/
  │   ├── bootstrap.js
  │   ├── bootstrap.min.js
  └── img/
      ├── glyphicons-halflings.png
      └── glyphicons-halflings-white.png
Following is the content of index.html 
<!DOCTYPE html>
<html>
  <head>
    <title>Bootstrap 101 Template</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet" media="screen">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>
